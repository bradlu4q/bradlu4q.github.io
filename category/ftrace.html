Ftrace 學習

參考資料：

必要

  • available_tracers - 列出目前支援的 tracer 類型

  • current_tracer - 目前使用的 tracer，基本上有下列的選擇

    hwlat blk mmiotrace function_graph wakeup_dl wakeup_rt wakeup function nop

    • trace - 輸出, 使用 cat command 印出

使用 tracer: function

### 列出目前支援 tracer 類型
root@bl4-u2204:/sys/kernel/debug/tracing# cat available_tracers 
hwlat blk mmiotrace function_graph wakeup_dl wakeup_rt wakeup function nop

### 指定使用 function 類型為 tracer
root@bl4-u2204:/sys/kernel/debug/tracing# echo function > current_tracer 

### 輸出結果 （取前20筆），可看到 "tracer: function"

使用 stack_trace_filter

### 找尋 kernel 內 "option_" 開頭的 function
root@bl4-u2204:/sys/kernel/debug/tracing# cat available_filter_functions | grep option_
acpi_rs_struct_option_length
pnp_option_priority_name
kgdboc_option_setup
gssx_dec_option_array [auth_rpcgss]
option_inquiry [usb_storage]
option_ms_init [usb_storage]
option_probe [option]
option_instat_callback [option]
option_release [option]
option_attach [option]

### 設定function: option_probe 為 stack_trace_filter,
root@bl4-u2204:/sys/kernel/debug/tracing# echo option_probe > stack_trace_filter

### 確認輸出狀態 目前為關閉 Stack tracer disabled
root@bl4-u2204:/sys/kernel/debug/tracing# cat stack_trace
        Depth    Size   Location    (0 entries)
        -----    ----   --------
#
#  Stack tracer disabled
#
# To enable the stack tracer, either add 'stacktrace' to the
# kernel command line
# or 'echo 1 > /proc/sys/kernel/stack_tracer_enabled'
#

### 開啟 stack tracer
root@bl4-u2204:/sys/kernel/debug/tracing# echo 1 > /proc/sys/kernel/stack_tracer_enabled

### 插入 RG502Q 印出 stack tracer 輸出
